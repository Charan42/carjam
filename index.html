🚗 CarJam — Blocked Vehicle Ranker

Upload a parking-jam image, trace the layout, and rank vehicles by how blocked they are. Rank 1 = most blocked.

  1. Upload the puzzle image.
  2. Set rows & cols, then Set grid area and drag a box over the parking grid.
  3. Auto-detect for a rough fill, or Add vehicle and drag across cells.
  4. Select a vehicle to set its arrow direction (the way it can move).
  5. Rank vehicles to see results.
